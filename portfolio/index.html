Tentang Saya

Nama lengkap saya adalah Kuncoro Buwono, saya lahir pada tanggal 6 juni 1994 di kota kembang yakni kota Bandung. saya biasa dipanggil kunto oleh keluarga dan teman dekat saya, saat ini usia saya berumur 25 tahun.

Saya ini adalah tipikal orang yang pemalu bila bertemu dengan seseorang yang baru dikenal, tetapi jika sudah mengenal lama dengan seseorang maka saya akan jauh lebih akrab. kemudian, pengalaman saya bekerja yakni pernah magang disebuah Koperasi dan Jobdesk saya di sana adalah sebagai System Analyst, kemudian saya juga pernah bekerja sebagai Software QA di sebuah perusahaan Start Up.

di luar pekerjaan, saya selalu melatih kemampuan saya entah dibidang editing video, pemrograman Frontend, mempertajam pengetahuan soal software QA, ataupun iseng-iseng membuat sebuah lagu.
